# 📔 My Diary Planner

A cute and interactive web application to manage your personal diary, schedule meetings, and organize tasks all in one place!

## Features ✨

- 📝 **Diary Entries**: Write, edit, and delete your daily diary entries
- 🤝 **Meeting Management**: Schedule and track your important meetings
- ✅ **Task Scheduler**: Create and manage your daily tasks with priorities
- 🎨 **Cute UI**: Beautiful and user-friendly interface
- 💾 **Data Persistence**: All data is saved to the database
- 📱 **Responsive Design**: Works on desktop and mobile devices

## Tech Stack 🛠️

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js, Express.js
- **Database**: SQLite
- **Styling**: Custom CSS with cute animations

## Installation 🚀

### Prerequisites
- Node.js (v14+)
- npm or yarn

### Setup Steps

1. Clone the repository
```bash
git clone https://github.com/Sajina366/my-diary-planner.git
cd my-diary-planner
```

2. Install dependencies
```bash
npm install
```

3. Start the server
```bash
npm start
```

4. Open your browser and visit `http://localhost:3000`

## Usage 📖

### Diary Section
- Click "New Entry" to create a diary entry
- Add date, mood, and your thoughts
- Save, edit, or delete entries as needed

### Meetings
- Manage all your meetings in one place
- Set meeting time, location, and participants

### Tasks
- Create tasks with priority levels (High, Medium, Low)
- Mark tasks as completed
- Filter tasks by status or priority
